# imai_atte
勤怠管理アプリ
![(<スクリーンショット 2024-04-13 16.58.43.png>)
]
##　作成した目的
スタッフが勤務開始終了、休憩開始休憩終了を打刻するだけで、勤務日毎のスタッフの勤務時間を把握するため

## アプリケーションURL
localhost

## 機能一覧
- 会員登録
- ログイン機能
- ログアウト機能
- 勤務開始
- 勤務終了
- 休憩開始
- 休憩終了
- 日付別勤怠情報取得
- ページネーション
- ユーザー一覧ページ
- ユーザー毎の勤務表一覧ページ

## 使用技術（実行環境）
- Laravel ８.75
